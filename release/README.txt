K Desktop Agent v0.1.0
======================

Claude AI 기반 Windows 데스크탑 자동화 에이전트

설치 방법
---------
1. K-Desktop-Agent_0.1.0_x64-setup.exe 더블클릭
2. "Windows의 PC 보호" 경고 시:
   - "추가 정보" 클릭 -> "실행" 클릭
3. 설치 마법사 따라가기
4. 시작 메뉴에서 "K Desktop Agent" 실행

필수 요구사항
-------------
- Windows 10/11 (64-bit)
- Git for Windows (https://git-scm.com/downloads/win)
- Python 3.10+ (https://python.org)
- Claude Max/Pro 구독 계정

최초 실행 시
------------
1. 앱이 자동으로 Claude 로그인 페이지를 엽니다
2. Claude 계정으로 로그인하세요
3. MCP 서버 연결 상태가 녹색으로 바뀌면 준비 완료!

사용법
------
- 자연어로 명령: "다운로드 폴더 정리해줘", "스크린샷 찍어서 분석해줘"
- Ctrl+Enter: 메시지 전송
- 시스템 트레이: 앱이 백그라운드에서 실행됨
